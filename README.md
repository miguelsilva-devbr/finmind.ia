# 🚀 FinMind
**Sua inteligência financeira, no seu WhatsApp.**
O FinMind é uma IA financeira criada para ajudar pessoas a controlar gastos, economizar dinheiro e tomar decisões financeiras melhores de forma simples, prática e direta.
Sem planilhas complicadas.  
Sem aplicativos confusos.  
Só conversa e orientação objetiva.

---

# 🧠 O Problema
Muitas pessoas:
- não sabem para onde o dinheiro vai
- gastam mais do que deveriam
- não possuem controle financeiro
- têm dificuldade para começar a investir
As ferramentas atuais geralmente são:
- complexas
- difíceis de usar
- pouco acessíveis

---

# 💡 A Solução
O FinMind simplifica a gestão financeira usando inteligência artificial.
O usuário conversa normalmente com a plataforma e recebe:
- 📊 análises financeiras
- ⚠️ alertas sobre erros financeiros
- 💰 sugestões de economia
- 🎯 próximos passos práticos
Tudo de forma simples e direta.

---

# ⚙️ Funcionalidades
- 📊 Análise automática de renda e despesas
- 💸 Registro inteligente de gastos
- 🧠 Memória financeira do usuário
- 📅 Histórico de despesas
- 💡 Sugestões financeiras personalizadas
- 🎯 Recomendações práticas de economia
- 📈 Orientação financeira para iniciantes
- 🔒 Backend seguro com API protegida
- ⚡ Respostas rápidas em tempo real
- 📱 Interface moderna e responsiva

---

# 🧩 Tecnologias Utilizadas
- **Frontend:** React + Next.js
- **Backend:** API Routes
- **Banco de Dados:** SQLite
- **IA:** Claude (Anthropic API)
- **Deploy:** Vercel
- **Estilização:** CSS + DM Sans

        ↓

Claude API + Banco de Dados
A chave da API permanece protegida no backend através de variáveis de ambiente.

🧠 Sistema de Memória Financeira
O FinMind possui um sistema de memória financeira capaz de:
salvar gastos do usuário
registrar valores e descrições
calcular despesas totais
recuperar histórico financeiro
acompanhar evolução financeira

Exemplo:
"Gastei R$50 no mercado"

O sistema salva automaticamente:
valor
categoria
data
histórico financeiro
Depois o usuário pode perguntar:
"Quanto eu gastei esse mês?"
E receber uma análise baseada no histórico armazenado.

📱 Experiência

👉 Web
